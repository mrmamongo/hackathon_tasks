Дано целое число n. Необходимо вывести количество чисел меньше n.

### Пример 1:
```
Ввод: n = 10
Вывод: 4
Explanation: There are 4 prime numbers less than 10, they are 2, 3, 5, 7.
```
### Пример 2:
```
Ввод: n = 0
Вывод: 0
```
### Пример 3:
```
Ввод: n = 1
Вывод: 0
```

### Ограничения:
```
0 <= n <= 5 * 106
```
